# Sample project with Symfony 4 and Docker
### Install

run docker:
```
 docker-compose up -d
```
get into the container:
```
 docker-compose exec php sh
```
install dependencies:
```
 composer install
```

### Start application

call localhost in your browser:
- [http://localhost](https://localhost/)
